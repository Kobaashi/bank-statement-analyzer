## Getting Started

First, run the development server:


# About implementation

It was a bit challenging working with PapaParse for the first time, but really interesting. I especially liked how the header option switches between an array of objects and an array of arrays. Vitest is also a great framework; I enjoyed learning how it works and how it outputs test results and errors.
